## GastroPerú
Proyecto educativo dedicado al desarrollo de una aplicación web para un portal de gastronomía

## Requisitos previos

Instalación de git, node y npm

## Instalación

### 1. Solicitar acceso al respositorio
### 2. Clonar repositorio en el entorno local
```bash
git clone [https://github.com/rmanasb10/gastroperu.git]
```
### 3. Navegar al directorio principal
```bash
cd gastroPeru
```
### 4. Instalar dependencias
```bash
npm install
```

## Ejecución

### 1. Construir entorno de desarrollo
```bash
npm parcel
```
### 2. Construir un entorno de producción
```bash
npm run build
```
### 3. Ejecutar entorno de producción
```bash
npm run start
```
